# FFmpeg

## Installation (linux, Mac)

| OS | Install command |
| --- | --- |
| MacOS | `brew install ffmpeg` |
| Ubuntu/Debian | `apt install ffmpeg` |
| RHEL/CentOS/Fedora | `dnf install ffmpeg` |

## Syntax

```bash
ffmpeg -i <input> [options] <output>
```
